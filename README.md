# Real-Time Restaurant Ordering System

## 📌 Description
A full-stack real-time restaurant ordering system with QR-based menu, live order tracking, and admin dashboard.

## 🚀 Features
- QR-based menu system
- Real-time order updates using Socket.io
- Customer ordering interface
- Reception dashboard for live tracking
- Admin panel for menu management

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MySQL (Railway)
- Real-time: Socket.io

## 🌐 Deployment
- Frontend: Netlify
- Backend: Render
- Database: Railway MySQL

## ▶️ Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
